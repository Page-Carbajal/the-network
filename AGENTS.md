The following project provides a set of API endpoints that simulate a social media platform.


## Tech Stack

- API and Backend: Deno2, Hono, Typescript, Faker.js, Zod, SQLite
- Frontend: HTML, Rect and Tailwind CSS, TypeScript

## Project Structure 

- src/server/index.ts: Main server file that sets up the API endpoints using Hono framework.
- src/data/: Directory containing mock data for users, posts, comments, likes, and followers
- src/data/db: SQLite database
- src/data/migrations: Database migration files
